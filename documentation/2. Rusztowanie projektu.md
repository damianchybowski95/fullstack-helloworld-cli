## Poniżej zapisuje podjęte kroki do budowania projektu.

1. Zainstaluj globalnie node, npm oraz typescript
   1.1. Node można zainstalować w chocolatey; wtedy instalacja zachodzi bardzo szybko.
   1.2. npm powinien zostać zainstalowany wraz z nodem. Można to sprawdzić komendą npm -v
   1.3. Typescript poleceniem `npm install -g typescript`
2. Uruchom `npm init -y`, aby utworzyć plik package.json.
3. Wykonaj `tsc --init`, aby wygenerować podstawową konfigurację typescript.
   3.1. Zmień type z commonjs, na module, by korzystać w pełni z typescripta
4. Dodaj rootDir, outDir oraz include src do pliku tsconfig.
5. Dodaj bin do package.json, które definiuje nazwę projektu po jego zainstalowaniu, oraz plik startowy projektu.
6. Dodaj skrypty do package.json:
   5.1. Build skrypt z poleceniem `tsc`
   5.2. polecenie `install_`, z podkreśleniem, ponieważ skrypt install w package.json jest magicznym poleceniem, więc install powinno działać jako `install_`.
   5.3. Skrypt start, który buduje, instaluje globalnie, oraz uruchamia globalne cli
7. Plik startowy index.ts powinien mieć w pierszsej lini kodu shebang: `#!/usr/bin/env node`
8. `npm install commander`
10. `npm install @inquirer/prompts` - Zasobnik zapytajników dla użytkownika
